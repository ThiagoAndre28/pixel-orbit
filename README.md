# Pixel Orbit

Projeto acadêmico desenvolvido durante o curso de Ciência da Computação.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- PHP
- SQLite

## Funcionalidades

- Sistema de cadastro e login
- Senhas armazenadas com segurança
- Jogo web em estilo pixel art
- Sistema de pontuação baseado em sobrevivência
- Dificuldade progressiva
- Ranking Top 10 persistente

## Objetivo

Aplicar conceitos de desenvolvimento web, banco de dados, lógica de programação e integração entre front-end e back-end.
